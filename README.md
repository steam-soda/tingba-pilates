# 挺拔 TINGBA · 居家垫上普拉提

针对**驼背、头前引、核心无力**的 30 分钟居家垫上普拉提 App。单文件、纯前端、离线可用，数据只保存在本机。专为 iPad 横屏使用设计（也适配手机竖屏）。

## 功能

- **今天**：每日自动从动作库挑选课程（按阶段配额 + 偏好 + 日期轮换，每天有变化）
- **日历**：月视图记录每天练了什么，连续天数 / 本周完成 / 累计统计
- **计划**：选每周练习日 + 提醒时间，可「同步到系统日历（.ics）」让 iPhone 收到当天推送
- **动作库**：35 个动作（含哑铃 / 弹力带 / 瑜伽砖器械动作），可标记「喜欢 / 少练」个性化课程
- **练习页**：动态简笔示范、圆环计时、防代偿警示 + 每 16 秒轮换的实时姿势提醒、随阶段切换的程序生成 lo-fi 背景音乐、中文语音播报
- 每个动作配「看 B 站示范视频」按钮（打开 B 站搜索该动作）

> ⚠️ 简笔动画只用于提示「身体朝向 + 动作节奏」，**不是标准动作示范**。正式练习请以文字要点和 B 站示范视频为准；负重动作不确定时先看视频或咨询教练。

## 使用

直接用浏览器打开 `index.html` 即可。iPad 上推荐用 Safari 打开后「分享 → 添加到主屏幕」，即可全屏运行，体验接近原生 App。

## 技术

- 单个 `index.html`，无依赖、无构建步骤
- 视觉遵循一套 spec-sheet 风格设计系统（纸灰底 / 墨色结构 / 蓝色 accent / 青柠点缀）
- SVG + SMIL 动画、Web Audio 实时生成音乐、Web Speech 语音、localStorage 存储
